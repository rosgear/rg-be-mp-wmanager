## <img src="https://raw.githubusercontent.com/rosgear/rg-be-mp-wmanager/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Расширение модуля «Менеджер виджетов»

Менеджер виджетов обеспечивает управление виджетами веб‑приложения RosGear, включая их установку, удаление и деинсталляцию.

Расширение модуля входит в состав Панели управления GePanel и cлужит расширением для [модуля «Маркетплейс приложений»](https://github.com/rosgear/rg-be-mp).

### Скриншот
<img src="https://github.com/rosgear/rg-be-mp-wmanager/blob/main/assets/help/grid.png?raw=true">

### Установка

Для добавления расширения модуля в ваш проект, вы можете просто выполнить команду ниже:

```
$ composer require rosgear/rg-be-mp-wmanager
```

или добавить в файл composer.json вашего проекта:
```
"require": {
    "rosgear/rg-be-mp-wmanager": "*"
}
```
или скачать архив на [странице расширения модуля](https://rosgear.ru/component/rg-be-mp-wmanager/) в каталоге приложений RosGear.

После добавления расширения модуля в проект выполните его установку в редакцию веб‑приложения с помощью Панели управления GePanel.

## <img src="https://raw.githubusercontent.com/rosgear/rg-be-mp-wmanager/refs/heads/main/assets/images/icon.svg" width="64px" height="64px" align="absmiddle"> Module extension «Widget Manager»

The Widget Manager allows you to manage widgets in the RosGear web application, including installing, removing, and uninstalling them.

The module extension is part of the GePanel Control Panel and serves as an extension for the [«Application Marketplace» module](https://github.com/rosgear/rg-be-mp).

### Screenshot
<img src="https://github.com/rosgear/rg-be-mp-wmanager/blob/main/assets/help/grid.png?raw=true">

### Installation

To add the module extension to your project, you can simply run the command below:

```
$ composer require rosgear/rg-be-mp-wmanager
```

or add to your project's composer.json file:
```
"require": {
    "rosgear/rg-be-mp-wmanager": "*"
}
```
or download the archive from the [module extension page](https://rosgear.ru/component/rg-be-mp-wmanager/) in the RosGear application catalog.

After adding the module extension to the project, install it into the web application edition using the GePanel Control Panel.
